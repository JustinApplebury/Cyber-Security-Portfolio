# Cybersecurity Portfolio

Welcome to my cybersecurity portfolio. This repository contains structured writeups, lab reports, and technical case studies from Capture the Flag challenges, cybersecurity training platforms, and home lab projects.

The purpose of this portfolio is to demonstrate how I approach cyber investigations, document technical findings, communicate risk, and translate hands-on analysis into professional reporting. Each writeup is written with an emphasis on clear methodology, evidence-based findings, defensive recommendations, and lessons learned.

## Portfolio Focus

This repository highlights practical skills across several cybersecurity domains:

* Digital forensics
* Threat intelligence
* SOC-style investigation workflows
* Malware indicator enrichment
* Network and host-based analysis
* MITRE ATT&CK mapping
* Incident response documentation
* Defensive recommendations and reporting
* Professional technical communication

## Reporting Style

My writeups are structured to reflect the type of documentation expected in professional cybersecurity environments. Where appropriate, reports include:

* **BLUF / Executive Summary** — concise summary of what happened, why it matters, and the final outcome
* **Scope and Objectives** — clearly defined investigation goals and constraints
* **Methodology** — step-by-step explanation of the analytical approach
* **Technical Findings** — evidence-supported observations and conclusions
* **MITRE ATT&CK Mapping** — alignment of observed behavior to adversary tactics and techniques
* **Defensive Recommendations** — practical steps for detection, mitigation, and hardening
* **Lessons Learned** — key takeaways for future investigations
